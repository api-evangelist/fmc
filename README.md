# FMC Corporation (fmc)

FMC Corporation is a Philadelphia-headquartered global agricultural sciences company that develops and supplies crop protection products — herbicides, insecticides, fungicides, and biological solutions — alongside precision agriculture technology. Originally founded as the Food Machinery Corporation, the modern FMC focuses on productivity and resilience for growers worldwide.

This repository tracks the FMC Corporation entity and any APIs, developer tools, and machine-readable artifacts. As of this revision no public APIs or developer programs have been documented.

Note that "FMC" is also commonly used as an abbreviation for the legacy FMC Technologies brand (now TechnipFMC), which is tracked separately under the [fmc-technologies](https://github.com/api-evangelist/fmc-technologies) repository.
